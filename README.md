# PokeRogueData
PokeRogueData is a browser extension designed to assist players while playing Pokerogue by automatically displaying data of the opposing Pok√©mon so you don't have to look it up yourself.
To make this project a browser extension: Clone this repository into a folder of your choice: git clone https://github.com/Khawly/PokeRogueData.git
                                          Go to chrome://extensions.
                                          Enable Developer mode.
                                          Click Load unpacked, then select your project folder.
                                          Refresh PokeRogue and you're all set!
